# Attendance Tracker

A simple but powerful attendance management web app.

Built to handle real-world messy data and make attendance tracking faster and easier.

---

## Features

- Upload student list from:
  - Excel (.xlsx)
  - PDF
  - DOCX

- Automatically extracts:
  - Roll numbers
  - Student names

- Mark attendance:
  - Present / Absent
  - Auto-mark remaining as absent

- Search students instantly

- Export reports:
  - Excel format
  - PDF format

- Advanced report system:
  - Generate Monthly reports from multiple files
  - Generate Semester analytics

- Works offline (PWA support)

---

## Tech Used

- HTML, CSS, JavaScript
- SheetJS (Excel handling)
- PDF.js (PDF parsing)
- jsPDF (PDF generation)
- Mammoth.js (DOCX parsing)
- Service Workers (Offline support)

---

## How to Use

1. Open the app
2. Fill basic details (faculty, subject, class, semester)
3. Upload student list file
4. Mark attendance
5. Export report

For advanced:
- Upload multiple files in "Master Reports"
- Generate monthly or semester reports

---

## Why I Built This

Managing attendance manually is slow and annoying.  
This project focuses on handling unstructured data and turning it into usable reports quickly.
